Nao pausar o pipeline para o usuario confirmar o Design Blueprint. Seguir direto do designer para specialist/developer.

**Why:** Usuario nao quer confirmar blueprint, so quer conferir a implementacao final. Pausa no blueprint atrasa o fluxo sem agregar valor.

**How to apply:** A unica pausa do pipeline e na fase 3 (implementacao). Blueprint e Saffron Guide seguem sem interrupcao.
