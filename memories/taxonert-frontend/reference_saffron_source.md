O repositório `saffron_design_system` está disponível localmente na mesma workspace, em `C:\repositories\saffron_design_system`.

**Why:** Quando o MCP, skills e o código já implementado no projeto não forem suficientes para entender a API de um componente Saffron (slots, eventos, properties, parts), o código fonte do design system é a fonte de verdade.

**How to apply:**
- Usar como último recurso após MCP tools, skills e grep no `node_modules`
- Útil para verificar: slots disponíveis, eventos custom, CSS parts, variantes, stories do Storybook
- Caminho: `C:\repositories\saffron_design_system`
